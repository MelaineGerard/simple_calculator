Présentation

Simple Calculator est une application web de calculatrice. Elle permet de faire des opérations simples comme l'addition, la soustraction, la multiplication et la division.

Elle a été réalisée par Melaine Gérard dans le cadre d'un projet de formation.


Utilisation

La calculatrice est composée d'un écran et d'un clavier.

L'écran affiche le résultat des opérations.

Le clavier permet de saisir les nombres et les opérateurs.

Les opérations sont effectuées de gauche à droite.

Les opérations sont prioritaires dans l'ordre suivant : multiplication, division, addition, soustraction.


Historique

L'historique des opérations est affiché à gauche de la calculatrice.

Il affiche toutes les opérations effectuées durant la session.


Technologies utilisées

Simple Calculator a été réalisée en HTML, CSS et JavaScript.
